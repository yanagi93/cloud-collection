## Layer Responsibilities

### API Layer

```text
internal/api/
в”ңв”Җв”Җ gen/
в””в”Җв”Җ handler/
```

* `gen/`

  * Generated by ogen
  * OpenAPI types
  * Server interfaces
  * Router

* `handler/`

  * Implements ogen generated interfaces
  * Converts HTTP requests/responses
  * Calls service layer

### Service Layer

```text
internal/service/
```

* Business logic
* Transaction control
* Validation
* Orchestration across repositories

### Repository Layer

```text
internal/repository/
```

* Wraps sqlc generated queries
* Database access abstraction
* Converts DB models if needed

### Database Layer

```text
internal/db/
в”ңв”Җв”Җ gen/
в””в”Җв”Җ postgres.go
```

* `gen/`

  * Generated by sqlc
  * Query methods
  * DB models

* `postgres.go`

  * Connection setup
  * Pool configuration
